# Teatralidad

Un sitio web realizado con Laravel y TailwindCSS para la Beca Neoris en Programación Full-Stack dictada por la Universidad Tecnológica Nacional.

## Tecnologías

-   PHP 7+
-   Laravel 5.7
-   TailwindCSS
-   Material Icons
-   Axios
-   MySQL
-   Git
